| `SlotGame` | `onResume` |
| `RouletteGame` | `onResume` |

## `roulette_spin.mp3` — звук вращения
Зациклен, играет пока барабаны / колесо крутятся. Останавливается когда результат получен.

| Экран | Условие запуска | Когда останавливается |
|-------|----------------|----------------------|
| `Flasher` | Старт вращения колеса | Ответ сервера получен |
| `RouletteResult` | Открытие экрана, краткий прокрут (≈1.2 сек) | Колесо встало на число |
| `SlotFlasher` | Старт барабанов | Ответ сервера получен |
| `SlotResult` | Открытие экрана, краткий прокрут (≈1.1 сек) | Барабаны встали |

Однократный, не зациклен (13.8 сек). Воспроизводится один раз до конца.

## Архитектура воспроизведения

- **SoundPool** (`bet`, `error`) — загружаются один раз при старте, воспроизводятся мгновенно. Нет задержки буферизации.
- **MediaPlayer** (остальные) — создаются через `MediaPlayer.create()` при каждом использовании, освобождаются в `onDestroy`.
- **AudioFocus** — запрашивается в `onResume`, освобождается в `onPause`/`onDestroy`. При потере фокуса: LOSS/LOSS_TRANSIENT → пауза, LOSS_TRANSIENT_CAN_DUCK → громкость 0.1, GAIN → громкость 1.0.
- **AudioHelper.isSoundEnabled()** — глобальный переключатель звука; все старты обёрнуты в проверку этого флага.
