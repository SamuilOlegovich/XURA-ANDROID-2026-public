Структура мемо

MemoData: CMD:param1:param2:...
MemoType: text/plain
MemoFormat: UTF-8
Разделитель : — короткий и не требует экранирования. Первый токен — команда, дальше параметры. Всё в верхнем регистре для надёжного парсинга.

Игра «Угадай цвет»

BET:REDСтавка на красное
Обычная ставка на цвет
→ BET:RED
BET:BLKСтавка на чёрное
Обычная ставка на цвет
→ BET:BLK
WIN:REDПобеда
Сервер → клиенту: выигрыш
→ WIN:RED
LOSEПроигрыш
Сервер → клиенту: проигрыш
→ LOSE
JKPTДжекпот
Лотто-выигрыш
→ JKPT

Рулетка — ставки

Формат для рулетки: RL:тип:значение:тип:значение:... — несколько ставок в одном мемо через повторение пар.
RL:N:17На число
Straight up — ставка на конкретный номер (0–36)
→ RL:N:17
RL:R / RL:BКрасное / Чёрное
Even-money ставка на цвет
→ RL:R или RL:B
RL:E / RL:OЧётное / Нечётное
Even / Odd
→ RL:E или RL:O
RL:L / RL:H1–18 / 19–36
Low / High
→ RL:L или RL:H
RL:D:1/2/3Дюжина
1-я, 2-я или 3-я дюжина
→ RL:D:1
RL:C:1/2/3Колонка
1-я, 2-я или 3-я колонка
→ RL:C:2
RL:SP:14-17Сплит
Ставка на два соседних числа
→ RL:SP:14-17
RL:ST:1-2-3Стрит
Три числа в ряд
→ RL:ST:1-2-3
RL:CR:1-2-4-5Корнер
Четыре числа углом
→ RL:CR:1-2-4-5
RL:6L:1-6Шесть строк
Six line — два стрита подряд
→ RL:6L:1-6
Несколько ставок в одном мемо: RL:N:17:R:B:D:2 → ставка на 17, на красное, на 2-ю дюжину. Парсятся попарно после RL:.

Системные команды

REFСтать рефералом
→ REF или REF:восстановление
REF:RECВосстановить реферала
→ REF:REC
DONПожертвование
→ DON
RFDВозврат средств
→ RFD

Лимит 200 символов — оценка заполнения

BET:RED7 / 200
RL:N:17:R:B:D:2 (3 ставки)18 / 200
RL:N:0:N:5:N:17:N:32:SP:14-17:CR:1-2-4-5:R:B:D:1 (8 ставок)~60 / 200
В 200 символах ASCII комфортно помещается 10–15 одновременных ставок на рулетке, включая сложные комбо.
