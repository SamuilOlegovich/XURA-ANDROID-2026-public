# XURA Slot Machine — Server Protocol

## 1. Что сервер получает от клиента

XRPL-транзакция на адрес игрового сервера (`NetworkConfig.SERVER_ROULETTE`).

**Поле MemoData (UTF-8, в транзакции hex-encoded):**
```
SLOT:{referral}
```

| Поле | Тип | Описание |
|------|-----|----------|
| `referral` | string | Реферальный код игрока или `"0"` если нет |

**Сумма ставки** — берётся из `Amount` входящей транзакции (в drops, делить на 1 000 000 → XRP).

**Примеры:**
```
SLOT:0           — без реферала
SLOT:REF123      — с реферальным кодом
```

---

## 2. Что сервер отправляет в ответ

XRPL-транзакция клиенту с выплатой (если WIN) и MemoData:

**Поле MemoData (UTF-8, hex-encoded):**
```
SLOT:{p0},{p1},{p2}:{WIN|LOSE}
```

| Поле | Тип | Описание |
|------|-----|----------|
| `p0` | int 0–83 | Стоп-позиция левого барабана |
| `p1` | int 0–83 | Стоп-позиция центрального барабана |
| `p2` | int 0–83 | Стоп-позиция правого барабана |
| `WIN\|LOSE` | string | Итог раунда |

**Сумма выплаты** — передаётся в `Amount` ответной транзакции (клиент читает из `delivered_amount`).
При LOSE транзакция не отправляется (или отправляется с нулевой суммой — на усмотрение сервера).

**Примеры:**
```
SLOT:45,66,22:WIN
SLOT:12,8,37:LOSE
```

| Индекс | Название | Глиф | Множитель выплаты | Кол-во на стрипе |
|--------|----------|------|-------------------|-----------------|
| `0` | XRP      | X    | ×2   | 30 |
| `1` | Rocket   | ▲    | ×5   | 20 |
| `2` | Moon     | ◑    | ×10  | 15 |
| `3` | Diamond  | ◆    | ×20  | 10 |
| `4` | Whale    | W    | ×50  | 5  |
| `5` | Jackpot  | ★    | ×250 | 1  |
| `6` | Wild     | ≋    | замещает любой | 3 |

**Wild** (индекс 6) не выплачивает сам по себе — только замещает любой другой символ при проверке линии.

- `LEFT`   = BASE

## 5. Как построить матрицу 3×3 из стоп-позиций

Каждый барабан показывает 3 символа (верхний / центральный / нижний).
Стоп-позиция — это индекс **центральной (средней) строки** на барабане.

```
top    = strip[(stop - 1 + 84) % 84]
middle = strip[stop]
bottom = strip[(stop + 1) % 84]
```

Итоговая матрица `matrix[row][col]`, где `row` 0=top, 1=middle, 2=bottom; `col` 0=left, 1=center, 2=right:

```python
for row in range(3):
    matrix[row][0] = LEFT  [(stopLeft   + row - 1) % 84]
    matrix[row][1] = CENTER[(stopCenter + row - 1) % 84]
    matrix[row][2] = RIGHT [(stopRight  + row - 1) % 84]
```

## 6. Линии выплат и подсчёт выигрыша

Проверяются **5 линий**:

| Линия | Ячейки матрицы |
|-------|----------------|
| Средняя (главная) | `[1][0]` `[1][1]` `[1][2]` |
| Верхняя           | `[0][0]` `[0][1]` `[0][2]` |
| Нижняя            | `[2][0]` `[2][1]` `[2][2]` |
| Диагональ ↘       | `[0][0]` `[1][1]` `[2][2]` |
| Диагональ ↗       | `[2][0]` `[1][1]` `[0][2]` |

**Правило линии:** три одинаковых символа (Wild замещает любой).
Алгоритм проверки одной линии `[a, b, c]`:
1. Убрать Wild (индекс 6) из линии
2. Если все оставшиеся символы одинаковы — линия выигрывает, символ = найденный
3. Если оставшихся нет (все Wild) — линия не выигрывает (специальный случай не предусмотрен)

**Выплата по линии:**
```
payout = bet * multiplier[symbol]
```

**Итоговая выплата** = сумма выплат по всем выигравшим линиям.
Если итого > 0 → WIN, отправить транзакцию с суммой выплаты.
Если итого = 0 → LOSE, транзакция не отправляется.

## 7. Алгоритм сервера (псевдокод)

```python
def handle_slot_bet(transaction):
    bet    = transaction.amount / 1_000_000      # drops → XRP
    memo   = decode_hex(transaction.memo_data)   # "SLOT:0" или "SLOT:REF123"
    parts  = memo.split(":")
    referral = parts[1] if len(parts) > 1 else "0"

    # Генерация
    stop_left   = random.randint(0, 83)
    stop_center = random.randint(0, 83)
    stop_right  = random.randint(0, 83)

    # Матрица 3×3
    matrix = build_matrix(stop_left, stop_center, stop_right)

    # Проверка 5 линий
    paylines = [
        [matrix[1][0], matrix[1][1], matrix[1][2]],  # средняя
        [matrix[0][0], matrix[0][1], matrix[0][2]],  # верхняя
        [matrix[2][0], matrix[2][1], matrix[2][2]],  # нижняя
        [matrix[0][0], matrix[1][1], matrix[2][2]],  # диагональ ↘
        [matrix[2][0], matrix[1][1], matrix[0][2]],  # диагональ ↗
    ]

    MULTIPLIERS = [2, 5, 10, 20, 50, 250, 0]
    total_payout = 0
    for line in paylines:
        sym = resolve_line(line)           # -1 если нет выигрыша
        if sym >= 0:
            total_payout += bet * MULTIPLIERS[sym]

    outcome = "WIN" if total_payout > 0 else "LOSE"
    memo_response = f"SLOT:{stop_left},{stop_center},{stop_right}:{outcome}"

    if total_payout > 0:
        send_xrp(
            to      = transaction.sender,
            amount  = total_payout,
            memo    = memo_response
        )

def resolve_line(line):
    WILD = 6
    found = -1
    for sym in line:
        if sym == WILD:
            continue
        if found == -1:
            found = sym
        elif found != sym:
            return -1   # два разных не-Wild → нет выигрыша
    return found        # -1 если все Wild (не выигрывает)

def build_matrix(stop_left, stop_center, stop_right):
    matrix = [[0]*3 for _ in range(3)]
    for row in range(3):
        matrix[row][0] = LEFT  [(stop_left   + row - 1) % 84]
        matrix[row][1] = CENTER[(stop_center + row - 1) % 84]
        matrix[row][2] = RIGHT [(stop_right  + row - 1) % 84]
    return matrix
```

## 8. Пример полного раунда

**Клиент → Сервер:**
```
Transaction:
  Destination: rSERVER_ADDRESS
  Amount:      1000000 drops (= 1 XRP)
  MemoData:    hex("SLOT:0")
```

**Сервер генерирует:**
```
stop_left=44, stop_center=16, stop_right=72
```

**Матрица (stop-1, stop, stop+1 для каждого барабана):**
```
        LEFT[43]=0  CENTER[15]=0  RIGHT[71]=0
        LEFT[44]=5  CENTER[16]=5  RIGHT[72]=5   ← JACKPOT на всех трёх!
        LEFT[45]=2  CENTER[17]=2  RIGHT[73]=2
```

**Линии:**
- Средняя: [5, 5, 5] → Jackpot × 250 → выплата 1 × 250 = 250 XRP

**Сервер → Клиент:**
```
Transaction:
  Destination: rCLIENT_ADDRESS
  Amount:      250000000 drops (= 250 XRP)
  MemoData:    hex("SLOT:44,16,72:WIN")
```
